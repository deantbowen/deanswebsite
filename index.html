About me

I have most recently served as a Cyber Security Analyst II, where I’ve been responsible for overseeing comprehensive threat analysis, managing detection and response systems (MDR), and strengthening security operations for the past 1 year and 6 months. In this role, I have specialized in key areas such as vulnerability management, web application security, and email security, ensuring strong defenses against evolving cyber threats. I’ve also played a key role in real-time monitoring and proactive threat hunting, consistently adapting to emerging security challenges.

I find it deeply rewarding to stay ahead of cybercriminals, using advanced tools and techniques to secure sensitive data and critical systems. The dynamic nature of cybersecurity challenges me to continuously learn, adapt, and improve, ensuring that I can make a meaningful impact in safeguarding the digital world.

Resources

TOP 10 free security tools:

Wireshark: Network protocol analyzer for capturing and analyzing network traffic.

Nmap: Network scanner for discovering hosts, services, and open ports.

Metasploit Framework: Penetration testing framework for developing and executing exploit code.

Snort: IDS/IPS for monitoring network traffic and detecting malicious activity.

Suricata: IDS/IPS for real-time network traffic analysis and threat detection.

OpenVAS: Vulnerability scanner for detecting and assessing security weaknesses.

Zeek (formerly Bro): Network monitoring framework for analyzing network traffic and detecting anomalies.

TheHive: Security incident response platform for managing and tracking incidents.

MISP: Threat intelligence platform for sharing and correlating security indicators.

OSSEC: Host-based IDS for monitoring and analyzing system logs and events.
